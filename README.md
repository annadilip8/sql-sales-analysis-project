# sql-sales-analysis-project

This small project demonstrates fundamental SQL skills: schema design, data insertion, and analytical queries.

## Files
- `schema.sql` → Database schema (Customers, Products, Orders)
- `data.sql` → Sample dataset inserts
- `queries.sql` → Useful analytical queries (total sales, top products, customer summaries, monthly revenue, unsold products)

## How to use
1. Create a new database (SQLite/Postgres/MySQL).
2. Run `schema.sql` to create tables.
3. Run `data.sql` to insert sample data.
4. Run queries from `queries.sql` to analyze the dataset.

## Skills demonstrated
- SQL DDL (CREATE TABLE), DML (INSERT)
- Joins, Aggregations, GROUP BY
- Basic date grouping and NULL handling
